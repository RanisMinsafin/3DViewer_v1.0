3DViewer_v1.0

About program

The viewer is implemented on the QT and OpenGL graphic libraries. It works with .obj format 3d-models.

How to start

To start using the viewer, you need to install it using the make install command. This command will install the program on the desktop to the 3DWiewer folder.

Uninstall

To uninstall the program, you can do it using the make uninstall command.
